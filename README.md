# WriteBoard — 本地白板应用

一款基于 Electron + React + Fabric.js 的本地桌面白板工具，适用于教学、讲题、笔记等场景。支持无限画布、压感笔迹、笔迹平滑、多种背景模板、演示模式、PDF 批量导出等功能。

## 安装与启动

### 环境要求

- Node.js ≥ 18
- npm ≥ 9

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后自动打开 Electron 窗口，renderer 层支持热更新；修改 `electron/main.ts` 后需重启。

### 打包构建

```bash
npm run build       # 仅编译，输出到 out/
npm run package     # 打包为可执行文件（不生成安装包）
npm run dist        # 打包并生成安装包（Windows: NSIS .exe，macOS: dmg）
```

> 安装包输出在 `dist/` 目录。

## 首次使用

首次启动时，应用会弹出存储路径选择对话框。选择或新建一个**专用文件夹**作为工作区，所有白板数据均保存在此处，方便备份或通过 Git 进行版本管理。

存储结构：

```
你选择的文件夹/
├── index.json          # 分类元数据
├── boards/
│   ├── {uuid}.json     # 每块白板的完整数据（含画布 JSON）
│   └── ...
└── images/             # 白板中插入的图片文件
```

> 每块白板独立存储为单独文件，多人协作时各白板互不干扰。

## 界面概览

```
┌─────────────────────────────────────────────────────┐
│  标题栏（拖动区域）                        [─][□][×]  │
├────────┬────────────────────────────────────────────┤
│        │  工具栏                                     │
│  侧边栏 ├────────────────────────────────────────────┤
│        │                                             │
│  白板   │            画  布（无限画布）               │
│  列表   │                                             │
│        │                                             │
│  分类   ├────────────────────────────────────────────┤
│        │  白板标题（左上角可编辑）                    │
└────────┴────────────────────────────────────────────┘
```

- **侧边栏**（左）：管理所有分类和白板
- **工具栏**（上）：绘图工具、画布操作、导出等
- **画布**（中央）：无限画布区域，支持平移与缩放

## 绘图工具详解

工具栏从左到右依次为：

| 图标位置 | 工具 | 快捷键 | 说明 |
|---------|------|--------|------|
| 1 | **选择** | `1` | 选中、移动、缩放、旋转、删除画布元素 |
| 2 | **画笔** | `2` | 自由手绘，支持数位板/触摸屏压感（笔迹粗细随压力变化） |
| 3 | **荧光笔** | `3` | 半透明宽笔迹，适合高亮标注 |
| 4 | **橡皮擦** | `4` | 白色覆盖擦除，可调节橡皮大小 |
| 5 | **文字** | `5` | 点击画布任意位置插入可编辑文本框 |
| 6 | **形状** | `6` | 矩形、椭圆、直线、箭头，按需点击下拉箭头切换子类型 |
| 7 | **激光笔** | `7` | 红色光点跟随鼠标移动，不在画布上留下任何痕迹，适合演示指引 |
| 8 | **轨迹笔** | `8` | 绘制彩色路径，松开鼠标后轨迹自动渐隐消失，适合演示走势 |

### 画笔 / 荧光笔选项

选中画笔或荧光笔后，工具栏右侧会展开选项面板：

- **颜色**：提供 8 种预设色块，点击切换
- **笔迹宽度**：提供 4 档预设尺寸，点击切换
- **笔迹平滑**（`✦` 按钮）：启用后，松开鼠标时自动对笔迹进行 Chaikin 角切割平滑处理，使手写线条更流畅自然

### 橡皮擦选项

选中橡皮擦工具后，工具栏右侧会展开大小选项：

- **4 个预设圆圈**：代表 10 / 20 / 40 / 80 px，点击快速切换
- **滑动条**：支持 4–200 px 连续调节

### 形状工具

点击形状按钮右侧的小三角展开子类型菜单：

- 矩形（`Rect`）
- 椭圆（`Ellipse`）
- 直线（`Line`）
- 箭头（`Arrow`）

选中子类型后，在画布上**拖拽**即可绘制。颜色与线宽同样在右侧选项面板中调整。

## 画布操作

| 操作 | 方法 |
|------|------|
| **平移** | `Space` + 鼠标拖动 / 鼠标中键拖动 |
| **缩放** | 鼠标滚轮（范围 0.1× ~ 10×） |
| **粘贴图片/文字** | `Ctrl+V`（支持剪贴板截图） |
| **拖放图片** | 将图片文件直接拖入画布 |
| **插入图片** | 工具栏 → 图片按钮 → 选择本地文件 |
| **锁定元素** | 选中元素后点击工具栏锁定按钮，锁定后无法移动/缩放，防止误操作 |
| **撤销** | `Ctrl+Z` / 工具栏撤销按钮 |
| **重做** | `Ctrl+Shift+Z` / `Ctrl+Y` / 工具栏重做按钮 |
| **清空画布** | 工具栏垃圾桶按钮（⚠️ 不可撤销） |

## 背景模板

点击工具栏右侧的**方格纸图标**展开背景选择菜单，每块白板可独立设置：

| 模式 | 效果 |
|------|------|
| **空白** | 纯白背景（默认） |
| **方格纸** | 30px 浅灰方格网格 |
| **横线纸** | 30px 等间距水平线 |
| **点阵纸** | 30px 间距圆点阵列 |

背景随画布平移/缩放自动无限平铺，不受画布位置影响。

## 白板与分类管理

### 白板操作

| 操作 | 方法 |
|------|------|
| **新建白板** | 侧边栏顶部 `+` 按钮（放入「未分类」），或分类标题旁的 `+` 按钮 |
| **重命名** | 双击白板名称，输入新名称后回车或失焦确认 |
| **删除** | 白板行右侧 `×` 按钮，或右键菜单 → 删除 |
| **移动到分类** | 右键白板 → 移动到… → 选择目标分类 |
| **拖拽归类** | 鼠标拖拽白板条目到目标分类区域 |
| **编辑白板标题** | 点击画布左上角的白板标题区域内联编辑 |

同一分类下的白板按**名称自然排序**（支持数字顺序，如「第1题 < 第2题 < 第10题」）。

### 分类操作

| 操作 | 方法 |
|------|------|
| **新建分类** | 侧边栏底部「+ 新建分类」按钮 |
| **重命名分类** | 双击分类名称 |
| **展开/折叠** | 点击分类行左侧的箭头图标 |
| **删除分类** | 鼠标悬停分类行 → 点击 `×` 按钮 |
| **批量导出 PDF** | 鼠标悬停分类行 → 点击文件下载图标（见 [PDF 导出](#pdf-导出)） |

## 演示模式

按 `F11` 或工具栏演示按钮进入演示模式：

- 隐藏标题栏和侧边栏，画布铺满全屏
- 工具栏自动隐藏，**鼠标移到顶部**时自动浮现
- 所有绘图功能保持可用
- 按 `Escape` 或再次 `F11` 退出演示模式

---

## 导入 JSON

点击工具栏**导入 JSON 按钮**（文件夹+箭头图标），支持两种格式：

1. **单画板格式**：包含 `canvasJSON` 字段的 JSON 文件，内容导入到当前白板
2. **工作区格式**：包含 `categories` 和 `boards` 数组的完整工作区 JSON，自动合并到当前工作区（分类和白板均去重处理）

## PDF 导出

### 导出单块白板

点击工具栏的 **PDF 导出按钮**（文件+下载箭头图标）：

- **已设置默认目录**：直接将当前白板导出为 `{默认目录}/{分类名}/{白板名}.pdf`，导出完成后自动在资源管理器中定位到该文件
- **未设置默认目录**：弹出「另存为」对话框，默认路径预填为 `桌面/{分类名}/{白板名}.pdf`，可自行修改

### 导出分类下全部白板

鼠标悬停左侧边栏的**分类标题行**，点击**文件下载图标**批量导出：

1. 应用自动读取该分类下所有白板（按名称排序）
2. 在浏览器中使用 Fabric.js 离屏渲染每块白板
3. 按 `{目标目录}/{分类名}/{白板名}.pdf` 依次保存
4. 导出完成后自动在资源管理器中打开目标文件夹
5. 导出期间图标显示旋转动画，完成前不可重复点击

> **目标目录选择规则**：优先使用设置中配置的默认目录；未配置时弹出文件夹选择对话框。

导出过程中图片路径会自动还原为本地文件路径，无需手动处理。

## 设置

点击工具栏右侧**齿轮按钮**打开设置面板，包含两个选项卡：

### 快捷键

- 列出所有可自定义的快捷键操作
- 点击任意快捷键按钮后按新组合键即可覆盖
- 修改后的快捷键以**橙色**显示以示区别
- 点击「恢复默认」一键还原所有快捷键

### 导出

配置 **PDF 默认导出路径**：

- 点击**浏览…**按钮选择目标文件夹
- 设置后，单板导出与批量导出均自动使用该目录，不再弹窗询问
- 点击**清除**可恢复为每次手动选择路径的行为
