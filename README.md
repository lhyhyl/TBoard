# WriteBoard — 本地白板应用

一款基于 Electron + React + Fabric.js 的本地桌面白板工具，适用于教学、讲题、笔记等场景。支持无限画布、压感笔迹、多种背景模板、演示模式等功能。

## 启动项目

### 环境要求

- Node.js ≥ 18
- npm ≥ 9

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后会自动打开 Electron 窗口，支持热更新。

### 构建

```bash
npm run build       # 编译到 out/ 目录
npm run package     # 打包为可执行文件（不生成安装包）
npm run dist        # 打包为安装包
```

## 首次使用

首次启动时需要选择一个**存储文件夹**，所有白板数据（包括画板 JSON、图片等）都保存在该文件夹中，方便通过 Git 同步或备份。

存储结构：

```
你选择的文件夹/
├── index.json          # 画板列表 & 分类元数据
├── boards/
│   ├── {id}.json       # 每块画板的完整数据
│   └── ...
└── images/             # 插入的图片文件
```

## 功能介绍

### 绘图工具

| 工具 | 说明 |
|------|------|
| **画笔** | 自由绘制，支持数位板压感（笔迹粗细随压力变化） |
| **荧光笔** | 半透明笔迹，适合标注重点 |
| **橡皮擦** | 白色覆盖擦除 |
| **形状** | 矩形、椭圆、直线、箭头，拖拽绘制 |
| **文字** | 点击画布插入文本框 |
| **激光笔** | 红色光点跟随鼠标，不留痕迹，适合演示指示 |
| **轨迹笔** | 绘制彩色轨迹线，松开后轨迹自动渐隐消失，适合标注路径 |
| **选择** | 选中、移动、缩放、旋转画布上的元素 |

### 画布操作

- **无限画布**：画布无固定边界，可向任意方向平移
- **平移**：按住 `Space` + 拖动，或鼠标中键拖动
- **缩放**：鼠标滚轮（支持 0.1x ~ 10x）
- **粘贴图片/文字**：`Ctrl+V` 或右键菜单粘贴
- **拖放图片**：将图片文件直接拖入画布
- **元素锁定**：选中元素后通过工具栏或右键菜单锁定，防止误触移动

### 背景模板

通过工具栏右侧的背景按钮切换：

- **空白**（默认）
- **方格纸** — 30px 灰色网格
- **横线纸** — 30px 水平线
- **点阵纸** — 30px 圆点阵列

背景会随平移/缩放自动无限平铺，保存在画板数据中。

### 画板管理

- 左侧边栏管理所有画板和分类
- 支持新建、重命名（双击）、删除画板
- 支持创建分类文件夹，拖拽画板归类
- 分类支持展开/折叠（小箭头）
- 每块画板可在左上角设置独立标题（点击编辑）

### 演示模式

按 `F11` 进入演示模式：

- 隐藏标题栏和侧边栏，画布全屏
- 工具栏自动隐藏，鼠标悬停时显示
- 按 `Escape` 或再次 `F11` 退出

### 导入导出

- 支持导入 JSON 文件（单画板 canvasJSON 或完整工作区格式）
- 所有数据以 JSON 文件存储，可直接用 Git 管理版本

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `1` | 切换到选择工具 |
| `2` | 切换到画笔 |
| `3` | 切换到荧光笔 |
| `4` | 切换到橡皮擦 |
| `5` | 切换到文字工具 |
| `6` | 切换到形状工具 |
| `7` | 切换到激光笔 |
| `8` | 切换到轨迹笔 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` / `Ctrl + Y` | 重做 |
| `Delete` / `Backspace` | 删除选中元素 |
| `Space + 拖动` | 平移画布 |
| `鼠标滚轮` | 缩放画布 |
| `F11` | 进入/退出演示模式 |
| `Escape` | 退出演示模式 |

## 技术栈

- **Electron 28** — 桌面应用框架
- **React 18** — UI 框架
- **Fabric.js 6** — 画布引擎
- **Zustand** — 状态管理
- **Tailwind CSS** — 样式
- **electron-vite** — 构建工具
- **TypeScript** — 类型安全
